# 🚀 InsightFlow AI

InsightFlow AI is a web application that analyzes customer feedback and provides structured insights. It also includes an app research feature to analyze popular apps and suggest improvements.

⚠️ Note: This project is a prototype built as our first project. Some responses may be simulated due to API limitations.
---

## 📌 Problem Statement

Product teams receive feedback from multiple sources, but analyzing it manually is time-consuming and inefficient. Important insights often get missed.

---

## 💡 Solution

InsightFlow AI solves this problem by:

- Analyzing feedback automatically  
- Identifying key issues and feature requests  
- Providing sentiment analysis (positive/negative/neutral)  
- Generating actionable recommendations  
- Researching other apps for better product decisions  

---

## ✨ Features

- 🔍 Feedback Analysis  
- 😊 Sentiment Detection  
- 💡 Feature Suggestions  
- 🚀 Recommendations  
- 📱 App Research Tool  
- ⚡ Works without API (offline mode)  

---

## 🛠️ Tech Stack

- Python (Flask)  
- HTML, CSS, JavaScript  
- Optional AI (Gemini API)
